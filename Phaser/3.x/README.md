## How to build
* $ `npm install`
* $ `npm run build`

## Make sure project structure like this:
```
Your project
    |-- libs
        |-- phaser.d.ts
    |-- node_modules
        |-- ...
    |-- out
        |-- ...
    |-- src
        |-- ...
    |-- ...
```
